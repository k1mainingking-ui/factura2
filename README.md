# Factura Beauty Boutique Telegram Bot

🤖 Telegram бот для премиального бутика красоты Factura в Новосибирске с интегрированным AI-ассистентом.

## 📋 Описание

Бот предоставляет клиентам бутика удобный интерфейс для:
- Просмотра услуг по категориям
- Получения контактной информации
- Консультаций с AI-ассистентом
- Получения информации о салоне

## ✨ Возможности

### 🔹 Главное меню
- 💅 **Услуги** — просмотр всех услуг по категориям
- 📍 **Контактные данные** — адрес, телефон, часы работы
- 💬 **Спросить AI-ассистента** — диалог с интеллектуальным консультантом
- ℹ️ **О салоне** — информация о бутике Factura

### 🔹 AI-ассистент
- Отвечает на вопросы об услугах
- Консультирует по уходу за кожей, волосами, ногтями
- Рекомендует подходящие процедуры
- Помогает выбрать услугу
- Сохраняет контекст диалога (последние 10 сообщений)
- Команда `/reset` для очистки истории

## 🛠️ Технологии

- Python 3.11+
- aiogram 3.x (асинхронный фреймворк)
- OpenAI API (через ProxyAPI/DeepSeek и т.д.)
- python-dotenv для переменных окружения
- Логирование в файл и консоль

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone <репозиторий>
cd factura-bot
```

### 2. Создание виртуального окружения
```bash
python -m venv venv
```

#### Активация на Windows:
```bash
venv\Scripts\activate
```

#### Активация на Linux/macOS:
```bash
source venv/bin/activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Настройка конфигурации
```bash
# Скопируйте пример конфигурации
copy .env.example .env
```

Отредактируйте файл `.env` и заполните реальные данные:
```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
AI_API_KEY=ваш_ai_api_ключ
AI_BASE_URL=https://api.proxyapi.ru/openai/v1
AI_MODEL=gpt-4o-mini
```

### 5. Запуск бота
```bash
python main.py
```

## 📁 Структура проекта

```
factura-bot/
├── main.py              # Точка входа, запуск бота
├── handlers.py          # Обработчики команд и сообщений
├── keyboards.py         # Все клавиатуры бота
├── ai_service.py        # Интеграция с AI-ассистентом
├── services_data.py     # Список услуг бутика
├── config.py            # Конфигурация и логирование
├── .env                 # Переменные окружения (создается вручную)
├── .env.example         # Пример конфигурации
├── requirements.txt     # Зависимости проекта
├── .gitignore           # Игнорируемые файлы
├── README.md            # Документация
└── bot.log              # Лог файл (создается автоматически)
```

## 🔒 Безопасность

- Все секретные ключи хранятся только в переменных окружения
- Файл `.env` игнорируется git'ом
- Логи не содержат конфиденциальной информации

## 📝 Примечания

- AI-ассистент не выдумывает цены и информацию, которой нет в базе
- При недоступности AI бот предлагает позвонить по телефону
- История диалогов хранится только в оперативной памяти и очищается при перезапуске бота
